# Volume and dz.

Volume and dz.

## Usage

``` r
ref_vol_dz
```

## Format

- variable:

  Name of the physical variable: "volume" and "dz".

- polygon:

  Boxid starting from 0 to numboxes - 1.

- layer:

  Layerid starting from 0 to numlayers - 1.

- time:

  Simulation time in years. Modeltimestep was converted to actual time
  based on the settings in the 'run.prm' file.

- atoutput:

  Obseravtion column storing the actual output value. volume in m^3 dz
  in m
