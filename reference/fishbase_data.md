# fishbase_data

A table of all the the species found in Fihsbase, including taxonomic
classification.

## Usage

``` r
fishbase_data
```

## Format

A data frame with 33104 rows and 12 variables:

- `SpecCode`:

  integer. Species code.

- `Genus`:

  character.

- `Species`:

  character.

- `SpeciesRefNo`:

  integer. Reference number.

- `FBname`:

  character. Fishbase name.

- `SubFamily`:

  character.

- `FamCode`:

  integer. Family Code.

- `GenCode`:

  integer. Genetic Code.

- `SubGenCode`:

  integer. Sub Genetic Code.

- `Family`:

  character.

- `Order`:

  character.

- `Class`:

  character.
